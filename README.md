# 君子不器 成己达人

 现在玩博客的都可以算是珍稀动物了吧  

 其实大多数人的博客，都应该算作是笔记，自己记录一些东西  

 如果恰好，别人用得到，那便是做了善事了  

 从小我就是个“聪明孩子”，学东西特别快  

 “聪明孩子”有个缺点，就是不求甚解  

 啥都想学，啥都会一点，但啥都不精  

 一个学水产养殖的，毕业之后竟然还干了好长时间的IT工作  

 现在干的是跟上面两个八百杆子都打不着的活儿  

 前段时间工作需要建一个说明文档，尝试了很多，最终还是搁置了  

 也可能不是最终  

 所以边学边做，断断续续的，也没什么头绪  

 最近小王让我帮他搭个博客，偶然邂逅了HUGO，惊为天人  

 这是什么神仙技法，太好用了吧  

 顺便建这个网站，记录一些学习过程、遇到的问题和解决方案  

 另外也记录一些学到的其他的奇奇怪怪的东西  

 说不准也写点随笔，毕竟当年也是个文艺青年  

 主要是自己看  

 当然如果对你有用，我觉得也是一件很棒的事儿  

 我又不是专业干这个的，所以如果觉得不好，也别喷  

 如果觉得好，那就拿着用  

 君子不器，成己达人  

 就这样
